## 核心功能

- **意图识别** - 精准理解租客问题，自动分类分流
- **知识库 + RAG** - 基于私有知识库回答，确保答案准确
- **7x24 在线** - 全天候即时响应，无需人工加班
- **工单智能路由** - 复杂问题自动创建工单并路由给客服

## 适用场景

- 物业公司 - 处理租客咨询、物业费、维修申报
- 房产中介 - 回答房源信息、合同问题、预约看房
- 长租公寓 - 租客服务、费用查询、投诉处理
